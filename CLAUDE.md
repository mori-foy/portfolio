# ポートフォリオサイト方針

ユーザー(日本語話者、グラフィックデザイナー志向)が個人ポートフォリオサイトを制作中(2026-07開始)。

## 方針

- 白ベース必須。パステル(淡い青・ラベンダー・ピンク・レモン・ミント)はあくまで差し色
- スクロール連動の動き・演出を重視、芸術的だが見やすさ優先
- **オリジナルキャラクターは廃止済み**(2026-07-06)。ピクセルアート→SVGベクターと2回実装したが最終的に全撤去。今後キャラは提案しない
- 代わりにジェネラティブアート路線。「動的」「高度」な表現を好む
- Heroは3回作り直した: フローフィールド→エディトリアル大タイポ→**WebGL版(現行)**。要望は「大きな文字を中心に置かない」「高難度でクオリティの高いかっこいいモーション」。現行構成:
  - GLSLドメインワープFBMの液体パステル背景(マウスで歪む)
  - ガラス質感ショートカットカード3枚(3D傾き・視差・光沢追従)
  - 左下に小さめのコピー(行マスクリビール)
  - 東京時間ライブ時計
  - カスタムカーソル(ドット+追従リング、サイト全体)
- PLAYGROUNDセクションは廃止済み(2026-07-07)。「川柳なう。」はWorksセクションのPickup(featured)カードとして統合。セクション構成は01 About / 02 Works / 03 Contact
