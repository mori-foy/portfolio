# Portfolio

グラフィックデザイナーのポートフォリオサイト。白ベース×パステルカラーを基調に、GLSLを使ったジェネラティブアート表現を取り入れています。

## 特徴

- 白ベース+パステル(淡い青・ラベンダー・ピンク・レモン・ミント)の配色
- WebGL(GLSLドメインワープFBM)による液体パステル背景、マウスで歪む
- ガラス質感のショートカットカード(3D傾き・視差・光沢追従)
- 東京時間ライブ時計
- カスタムカーソル(ドット+追従リング)
- PLAYGROUNDセクション: シード乱数によるジェネラティブ・ポスター(クリックで再生成)

## 構成

```
index.html     ページ本体
css/style.css  スタイル
js/main.js     WebGL/インタラクション実装
```

デザイン方針の詳細は [CLAUDE.md](./CLAUDE.md) を参照してください。

## ローカルでの確認

`index.html` をブラウザで開くだけで動作します。
